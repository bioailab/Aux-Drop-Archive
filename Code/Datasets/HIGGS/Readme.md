# HIGGS 
 - Original Source: https://archive.ics.uci.edu/ml/datasets/HIGGS . It contains 11M datapoints and 28 features.
 - We use only the first 1M data points and first 21 features.
 - Data: The preprocessed datasets can be found in - https://figshare.com/s/0cd0d6ad4d30a9e91e9a. Create a folder here called `data`. Save the dataset in the link in the `Code/Datasets/HIGGS/data/` folder.
 - Mask: The masking for all the experiments with the HIGGS data can be found here - https://figshare.com/s/644fe204eb591e104184. Create a folder here called `mask`. Save all the file from the link to the `Code/Datasets/HIGGS/mask/` folder.
